# Residenzen Nesselbach – Vite + Tailwind

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and fill `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. Start dev server: `npm run dev`

The page lives in [src/App.tsx](src/App.tsx) and uses Tailwind styles from [src/index.css](src/index.css).
